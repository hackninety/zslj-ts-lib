# 占事略決古法引擎 · 起课算法说明

> 本文是古法引擎的算法规范（引擎已独立为 `zslj-ts-lib` 库，源码见其 `src/`），逐条对照《占事略決》原文（章节见[全书合订本](../book/senji-ryakketsu.md)）。
> 引擎只实现书中规则；书中存疑处的处理见[存疑清单](textual-issues.md)，**不以后世通行六壬规则补洞**。

        │

        ▼

## 1. 历法层（calendar.ts）

- 四柱取 tyme4ts `SixtyCycleHour`：年以立春、月以节气、日界 23:00 起子时。
- **月将双模式**：
  - `zhongqi`（默认）：太阳过宫。取最近一个**中气**：雨水→亥（徵明）、春分→戌、谷雨→酉、小满→申、夏至→未、大暑→午、处暑→巳、秋分→辰、霜降→卯、小雪→寅、冬至→丑、大寒→子。依据补注01「月將者，太陽宮也」。
  - `yuejian`：按月建六合配将（正月建寅将亥……），即 senji05 字面十二月将表；复现古书「假令」课例用此模式。
- **旦暮界（古法）**：占时支在寅~酉为旦、戌~丑为暮（senji03「從寅主酉為旦，從戌至丑為暮」）。注意通行体系为卯~申昼，此为占事略決特色。

## 2. 天地盘（tianpan.ts）

「常以月將加占時」（senji01）：把月将支置于地盘占时支之上，余支顺布。

- `offset = (月将 − 占时) mod 12`；地盘支 g 上的天盘支 = `g + offset`。
- `offset = 0` 为**伏吟**、`offset = 6` 为**返吟**。

## 3. 四课（sike.ts，senji01 / senji08）

| 课 | 上神 | 下 |
|---|---|---|
| 一課（日之阳） | 日干寄宫上神 | 日干（克应以干五行论） |
| 二課（日之阴） | 一课上神本位所得之神 | 一课上神 |
| 三課（辰之阳） | 日支上神 | 日支 |
| 四課（辰之阴） | 三课上神本位所得之神 | 三课上神 |

寄宫（senji08）：甲寅、乙辰、丙巳、丁未、戊巳、己未、庚申、辛戌、壬亥、癸丑。

克应：`下克上 = 下賊上`（深）、`上克下 = 上剋下`（浅）。

## 4. 三传 · 課用九法（sanchuan.ts，senji02）

按原文九条顺序判定；**盘面状态（伏吟/返吟/八专日）优先分派**，因为它们的中末传规则不同：

### 判定顺序

1. 天地伏吟 → 第七法
2. 天地返吟 → 第八法
3. 五柔日（甲寅/庚申/己未/丁未/癸丑）→ 第九法
4. 其余 → 第一~六法依次

### 第一 賊剋
下贼上优先；无贼取上克下。去重规则：同神同位**且同克应**的课合并（八专日一课下为干、三课下为支，五行可异，故 key 必须含克应）。

### 第二 比用
多课竞用时取「与今日比」者：罡日（阳干日）取**阳支**上神、柔日取**阴支**上神。原文以十二月将名列举：罡日比日=神后功曹天罡勝先傳送河魁（子寅辰午申戌），柔日比辰=大吉大衝太一小吉從魁徵明（丑卯巳未酉亥）。

### 第三 涉害（古法）
俱比/俱不比时：**加孟为深（寅申巳亥）、加仲为半（子午卯酉）、加季为浅（辰戌丑未）**，取深者。
⚠️ 不采用后世「涉归本家计克数」法——那是《六壬大全》系的改订。

### 第四 先擧
涉害俱等：「日為先，辰為後，陽為先，陰為後」→ 引擎按课序一→四取先（一课=日之阳最先）。

### 第五 遙剋
四课无上下克时，取二三四课上神与日干的遥克：**神克日优先于日克神**；多者复用比、涉害裁决。

### 第六 昴星
无克亦无遥克：
- 罡日：初=地盘**酉宫上神**；中=辰上神、末=日上神。
- 柔日：初=天盘**酉（從魁）所临地盘支**；中=日上神、末=辰上神。

### 第七 伏吟
- 有克（乙癸日）：初传按贼克流程。
- 无克：罡日初=日上神、柔日初=辰上神。
- **中传=用神之刑、末传=用神之冲**（「其刑神為二傳，其衝神為三傳」两个「其」皆读作用神；另一读法见选项 `fuyinMo: 'chongOfZhong'`）。
- 用神自刑（辰午酉亥）时中传同初，原文未及，照字面处理并在判定路径中注明。

### 第八 返吟
- 有克：初按贼克流程，中末如常（返吟下自然形成 初-冲-初）。
- 无克（仅丁己辛之丑未六日）：依原文专条——**未日太一（巳）临亥为用、丑日徵明（亥）临巳为用**（实与丑未日驿马一致）；中传=辰上神、末传=日上神（「傳辰上，終日上神」）。

### 第九 五柔（八专）
- 有克：初按贼克流程，「其三傳如常」。
- 无克：罡日自日上神**顺数三神**（+2）为用；柔日自辰上阴神（四课上神）**逆数三神**（−2）为用。
- 中末传俱为日辰上神；若用即起日辰上，「唯有一傳」（独传标记 `singleChuan`）。

### 通例中末传（senji01）
「發用神為一傳，用神之本位所得二傳，二傳之神本位所得神為三傳」——中=天盘（初）、末=天盘（中）。

## 5. 十二天将（tianjiang.ts，senji03 / senji04）

### 贵人（天一）
| 日干 | 旦治 | 暮治 |
|---|---|---|
| 甲戊庚 | 大吉（丑） | 小吉（未） |
| 乙己 | 神后（子） | 傳送（申） |
| 丙丁 | 徵明（亥） | 從魁（酉） |
| 辛 | 勝先（午） | 功曹（寅） |
| 壬癸 | 太一（巳） | 大衝（卯） |

（「六、辛」之「六」京大本无，从校勘；庚已含于甲戊庚组。）

### 顺逆（背天門向地戶）
原文四例（在亥子前、在戌酉前、在辰巳前、在巳辰前）归纳：**贵人临地盘亥子丑寅卯辰顺布，临巳午未申酉戌逆布**。

### 环序
贵→腾虵→朱雀→六合→勾陣→青龍→天空→白虎→大裳→玄武→大陰→天后（「前盡於五，後終六」，前一腾虵、后一天后）。名号从原书：腾虵、勾陣、大陰、大裳、天一。

## 7. 卅六卦（gua36.ts，senji26）

36 个课体判定器，一盘可命中多卦。判定可信度分三级：

- **exact（依文可判）**：元首、重審、傍茹、蒿矢、寅視、伏吟、反吟、狡童逃女、惟薄不脩、亂首、贅婿寓居、陞跎、玄胎四牝、聯茹、曲直、炎上、稼穡、從革、潤下、無祿、絕紀、五憤四殺、鑄印乘軒
- **approx（近似判定）**：三交、龍戰、陰陽無親、九醜、天網、三光三陽、高蓋駟馬、斲輪織綬、斬關、天獄（原文条件有歧义或需外部信息，逐条见存疑清单）
- **info（总论归类）**：氣類物、新故
- **不判**：無婬（第十，文义过简，仅存原文）
